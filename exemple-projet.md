# GreenLoop — Marketplace B2B de matériaux recyclés pour le BTP

## Le problème

En France, le secteur du BTP génère **46 millions de tonnes de déchets par an**, dont seulement 35% sont effectivement recyclés. Les entreprises de construction qui souhaitent utiliser des matériaux recyclés font face à trois obstacles majeurs :

1. **Pas de visibilité** : il n'existe aucune plateforme centralisée pour trouver des fournisseurs de matériaux recyclés à proximité
2. **Manque de confiance** : aucune certification standardisée sur la qualité des matériaux de seconde vie
3. **Logistique fragmentée** : organiser le transport entre déchetteries, recycleurs et chantiers est un cauchemar administratif

Les artisans et PME du BTP perdent en moyenne 12h par mois à chercher des matériaux recyclés par téléphone et bouche-à-oreille.

## La solution

GreenLoop est une marketplace B2B qui connecte les recycleurs de matériaux (béton concassé, bois revalorisé, métaux, isolants biosourcés) avec les entreprises de construction.

### Fonctionnalités clés

- **Catalogue géolocalisé** : recherche de matériaux disponibles dans un rayon configurable autour du chantier
- **Fiches produit certifiées** : chaque lot est accompagné d'un certificat de conformité (normes NF, marquage CE quand applicable)
- **Devis instantané** : calcul automatique du prix livré incluant le transport optimisé
- **Suivi de traçabilité** : de la déchetterie au chantier, chaque matériau a un historique complet (origine, traitement, analyses)
- **Tableau de bord impact** : chaque achat affiche les tonnes de CO2 évitées et le taux de circularité du chantier

## Cible

- **Acheteurs** : artisans, PME du BTP (10-200 salariés), maîtres d'oeuvre engagés dans la RE2020
- **Vendeurs** : centres de tri, recycleurs agréés, déconstructeurs
- **Marché adressable** : 15 000 entreprises de construction en Île-de-France (lancement), 85 000 en France

## Proposition de valeur unique

GreenLoop est le seul acteur à combiner marketplace + certification qualité + logistique intégrée pour les matériaux recyclés du BTP. Nous transformons un marché opaque et fragmenté en une expérience d'achat aussi simple qu'une commande sur un site e-commerce classique.

## Modèle économique

- **Commission** : 5-8% sur chaque transaction (payée par le vendeur)
- **Abonnement Pro** : 149€/mois pour les fonctionnalités avancées (API, rapports RSE automatisés, priorité logistique)
- **Certification** : 299€ par lot certifié via notre réseau de laboratoires partenaires

**Objectif** : 2M€ ARR à 18 mois post-lancement.

|--------|--------------------------|
| Cycle Up | Focalisé sur le réemploi (mobilier, portes). GreenLoop cible les matériaux bruts (béton, bois, métal) |
| Backacia | Marketplace généraliste sans certification qualité intégrée |
| StockPro | Déstockage de matériaux neufs, pas de recyclé |

## Stack technique envisagée

- Frontend : application web responsive (React/Next.js)
- Backend : API REST (Node.js)
- Base de données : PostgreSQL avec extension géospatiale PostGIS
- Hébergement : Scaleway (souveraineté données FR)
- Paiement : Stripe Connect pour le marketplace
- Cartographie : Mapbox pour la géolocalisation des stocks
